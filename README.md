# AI Resume Analyzer

This project analyzes resumes using NLP and Machine Learning.

## Features
- Extract text from PDF
- Clean and process text
- Extract skills
- Calculate resume score
- Suggest missing skills

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python main.py
